# Задание 1

### Текст задачи

Дерево содержит вещественные числа. Округлить каждый элемент.

### Алгоритм решения

1. попадаем в main задаем константой значение размер дерева и пул значений для рандомных значений 
2. попадем в создание дерева после запускаем рандомное значение типа float так же проверяем что НАЧАЛЬНОЕ значение не равно 0  
3. после передаем в функцию вставки в дерево со значением -1 (чтобы считать кол значений ) 
4. в функции вставки дерева проверяем на наличее дерева если есть то смотрим по провалам дерева меньше в лево больше или равно право если же нет дерева первое дерево будет явялеться корнем этого дерева
5. так проходим столько раз сколько значений задано изначально
6. из мейна передаем в функцию мап которая округляет значения дерева тем что забирает значение и отправыляет в новое дерево 
7. выводит дерево
