# Задание 1

### Текст задачи

Дерево содержит вещественные числа. Округлить каждый элемент.

### Алгоритм решения

1. попадаем в main задаем константой значение размер дерева и пул значений для рандомных значений
2. попадем в создание дерева после запускаем рандомное значение типа float так же проверяем что НАЧАЛЬНОЕ значение не равно 0
3. после передаем в функцию вставки в дерево со значением -1 (чтобы считать кол значений )
4. в функции вставки дерева проверяем на наличее дерева если есть то смотрим по провалам дерева меньше в лево больше или равно право если же нет дерева первое дерево будет явялеться корнем этого дерева
5. так проходим столько раз сколько значений задано изначально
6. из мейна передаем в функцию мап которая округляет значения дерева тем что забирает значение и отправыляет в новое дерево
7. выводит дерево

# Задание 2

### Текст задачи

Сколько раз входит заданный элемент в дерево?

### Алгоритм решения

1. попадаем в main задаем константой значение размер дерева и пул значений для рандомных значений (так-же в рандомайзре строки задаем строку из каких значений будет состоять строка)
2. попадем в создание дерева после запускаем рандомайзер строк и передаем рандомную строку
3. после передаем в функцию вставки в дерево со значением -1 (чтобы считать кол значений )
4. в функции вставки дерева проверяем на наличее дерева если есть то смотрим по провилам дерева меньше в лево больше или равно право (для строк т.к у каждого символа есть свой id (циферки) они сранивают между собой и от этого дерево растет) если же нет дерева первое дерево будет явялеться корнем этого дерева так проходим столько раз сколько значений задано изначально
5. после созданного дерева запрашиваем у пользователя какое значение будем искать в дереве
6. полученное число с созданным деревом передаем в функцию по поиску вхождений
7. попадаем в функцию и смотрим пустое ли дерево
8. если дерево содержит значения то сначала проверяем левую "ветку" и так рекурсивно доходим до последнего листа
9. понимаем что конец по левой ветки возвращаем это под дерево смотрим есть ли правая ветка если есть то доходим по ней до конца, если конец попадаем в блок if и сверяем необходимой целью/значением
10. так проверяем и со всеми и складываем и возвращаем через аккумулятор фолд 
11. выводим красивое дерево и сколько схождений
